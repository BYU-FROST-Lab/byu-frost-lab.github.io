# About

Madeline is interested in robotics and specifically underwater robotics. Madeline has many recreational hobbies such as mountain biking, skiing, running, rock climbing, and scuba diving


# Research

Madeline is working on the multi robot terrain aided navigation project. The goal of this project is to build several underwater robots that can communicate with each other in order to more effectively navigate their way through underwater environments. Madeline's specific role is working on the low level control system which is responsible for directly driving the hardware. Madeline is also working on implementing several types of sensors for the robots.
